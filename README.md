# Autonomous Pet-Tracking Robot (RobotServer)
A custom-built, autonomous robot designed to track and monitor pets using computervision, LLM reasoning, and LiDAR SLAM navigation.  
### The Problem:
**The Limitation:** Most pet cameras just sit in one spot. They don't move, and they don't typically alert you when something important is happening, they strictly stream video. 

**The Goal:** I wanted to create something more active. A dynamic system capable of real-time identification, tracking, and spatial navigation, within a home environment.
